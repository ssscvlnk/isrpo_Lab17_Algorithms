## Что такое алгоритм?

**Алгоритм** — это точная конечная последовательность действий, направленная на достижение определённой цели или решение конкретной задачи.

### Свойства алгоритма:

- **Дискретность** — алгоритм состоит из отдельных шагов
- **Детерминированность** — каждый шаг однозначно определён
- **Конечность** — алгоритм завершается за конечное число шагов
- **Массовость** — применим для целого класса задач
- **Результативность** — всегда приводит к результату

## Что такое блок-схема?

**Блок-схема** — это графическое представление алгоритма, в котором каждый шаг изображается в виде геометрических фигур, соединённых стрелками.
